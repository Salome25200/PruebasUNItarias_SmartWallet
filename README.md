# PruebasUNItarias_Smart Wallet

**Nombre:** Priscila Salome

## Descripción 
Esta práctica es una simulación en Java de una billetera inteligente que gestiona el saldo de un usuario mediante depósitos y retiros. 
Tiene reglas como otorgar un 1% de cashback en recargas mayores a $100, bloquear operaciones sin fondos y limitar el saldo máximo a $5,000 para usuarios
estándar.

## Ejecución de Pruebas

Para ejecutar las pruebas unitarias de este proyecto, utilizamos Maven. 
Desde la terminal se ejecuta el siguiente comando:

```bash
mvn test
